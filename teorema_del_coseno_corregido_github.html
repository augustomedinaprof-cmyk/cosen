Ejercicio: calcular el lado desconocido

Escribí el resultado redondeado a 2 decimales.

Cómo resolver

Para calcular el lado desconocido a, se reemplazan los datos en: a² = b² + c² − 2bc · cos(A). Luego se calcula la raíz cuadrada.
